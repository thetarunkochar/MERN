# MERN
  - Ports
    - Server : 3001
    - Client : 3000
  
  - Commands to run locally 
    - Running server first(on 3001 port)
      - npm i 
      - nodemon index.js
    - Running the client side (on 3000 port)
      - npm i
      - npm start
  
  - Tech Stacks
    - Node Js
    - ReactJS
    - Mongoose
    - MongoDB
    - Redux
